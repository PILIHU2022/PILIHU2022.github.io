为每个文件重新设定最后修改时间

这一步其实是挺重要的，Hexo框架生成每篇文章的最后修改时间的依据是该文件的最后修改时间，而对于 Github Action 的容器来说，每一个文件都刚刚被下载下来，都是最新的，这就会导致你的每一篇文章每次部署时都会被认为刚才修改过。
我们这边可以直接使用 git 记录的时间来作为文件的最后修改时间。
